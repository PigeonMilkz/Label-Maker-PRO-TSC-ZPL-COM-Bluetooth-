# Label Maker PRO - Simple Setup Guide

This guide explains how to set up your label printer and use the software in very simple steps.

---

# 1. Turn On the Printer

1. Press the power button on the printer.
2. Wait until the light is steady (not flashing).
3. Make sure label paper is loaded correctly.

---

# 2. Connect Printer to Windows (Bluetooth)

If your printer uses Bluetooth

1. Turn on Bluetooth on your PC

    Click Start Menu
    Go to Settings
    Click Bluetooth & devices
    Turn Bluetooth ON

2. Put printer into pairing mode

    Hold the Bluetooth  Pair button on the printer for 3–5 seconds

3. On your computer

    Click Add device
    Choose Bluetooth
    Select your printer name
    Click Pair  Connect

4. Wait until it says Connected

---

# 3. Find Your Printer COM Port (IMPORTANT)

If you are not sure which COM port to use, follow this

### Method 1 (Easiest - In the App)

1. Open Label Maker PRO
2. Look at the Printer Port dropdown
3. Try each option (example COM3, COM4, COM5)
4. Click Print Test Label to see which one works

---

### Method 2 (Windows Device Manager)

1. Right-click the Start Menu
2. Click Device Manager
3. Scroll down and open Ports (COM & LPT)
4. Look for something like

    Bluetooth Serial Port (COM4)
    USB Serial Device (COM3)
5. The number in brackets is your COM port

Example

 COM4 = Bluetooth printer
 COM3 = USB printer

---

# 4. Select Printer Port in the App

1. Open Label Maker PRO
2. Go to Printer Port dropdown
3. Select the COM port you found (example COM4)

---

# 5. Print a Test Label

1. Type text into the first box
2. Click Refresh Preview
3. Click Print

---

# 6. If Nothing Prints

Try these simple fixes

 Make sure printer is ON
 Make sure Bluetooth says Connected
 Try a different COM port
 Restart printer and app
 Check paper is loaded correctly

---

# 7. Basic Tips

 Keep printer close to PC (Bluetooth works best nearby)
 Don’t move printer while printing
 Wait a few seconds after pressing Print

---

# Done!

You are now ready to print labels.

If anything goes wrong, restart the printer and try again.
